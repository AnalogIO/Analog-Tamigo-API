# Analog-Tamigo-API

The project requires a configuration file `AppSettingsSecrets.config` with the following format:

```xml
<?xml version="1.0"?>
<appSettings>
  <add key="TamigoUsername" value="USERNAME_TO_TAMIGO"/>
  <add key="TamigoPassword" value="PASSWORD_TO_TAMIGO"/>
</appSettings>
```
